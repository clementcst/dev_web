LA CALCULETTE :



  • Les angles pour les calculs de sin/cos/tan doivent être rentrés en radians.
  • Les opérations avec sin/cos/tan/log nécéssitent une paranthèse fermante pour fonctionner
  • Pour allumer la calculatrice, vous devez appuyer sur le bouton "on", si le bouton est rouge c'est que la calculatrice est éteinte, et à l'inverse si il est vert c'est qu'elle est allumée.
